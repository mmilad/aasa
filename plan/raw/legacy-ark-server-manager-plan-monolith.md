## 0. System Foundations

### 0.1 Runtime / Host Layer

- Process Management
  - Create abstraction layer for process execution
    - start(command, args, options)
    - stop(pid)
    - restart(pid)
  - Capture stdout/stderr streams
  - Emit lifecycle events
    - onStart
    - onExit
    - onError
  - Maintain process registry
    - serverId → PID mapping

- File System Layer
  - Define structure
    - /data/servers/{serverId}/
      - binaries/
      - configs/
      - mods/
      - logs/
      - backups/
  - Implement utilities
    - ensureDir(path)
    - readFile/writeFile
    - copyDir/deleteDir
  - Log streaming
    - tail logs
    - expose via API/WebSocket

- Networking
  - Port allocation service
    - check availability
    - reserve ports
  - RCON client
    - connect(host, port, password)
    - send(command)
    - handle timeouts/retries
